# excel-portfolio
This repository showcases 6 Excel projects demonstrating data analysis, pivot tables, Power Query transformations, dashboards, and decision modeling.


## Projects
1. Gradebook Tracker – logical formulas, conditional formatting, summary calculations
2. Sales Database – multi-sheet dataset with analysis and pivot tables
3. Mini Projections – financial projection and scenario modeling
4. Job Decision Model – weighted scoring and decision support
5. Pivot Table Analysis – summarization and visual insights
6. Power Query Transformation – data cleaning and reshaping pipelines
